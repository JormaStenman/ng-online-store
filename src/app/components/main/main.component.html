Demo Online Store

This web application demonstrates the following features usually found in any online store website:

  • A list of products can be browsed.
  • The details of any product can be viewed.
  • Products can be added into a shopping cart.
  • The contents of the shopping cart can be manipulated before ordering.
  • The list of orders can be viewed.
  • The details of any order can be viewed.
  • An order can be cancelled.
  • Product inventories change when orders are placed or cancelled.
  • A navigation bar is always visible at the top of the page.
